Para compilar se usará javac 21.0.4    en openjdk 21.0.4

Abrir la terminal a la altura del Makefile
Después, seguir las instrucciones dadas en pantalla

Para compilar los archivos, escribir "make"
Para limpiar, escribir "make clean"
Para ejecutar el archivo, escribir "make run"

Se descubrira un nuevo planeta cuando el jugador haya visitado todos los planetas ya descubiertos al menos una vez

Al momento de elegir un item para comprar de un asentamiento, no se pide el numero del item segun orden, se pide el
numero del item segun el tipo que sea, por ejemplo para
● - Propulsor MK 2
● - Escudo MK 1
Hay que ingresar 2 para comprar el propulsor MK2, y 5 para comprar el Escudo
Los numeros de cada item a comprar se detallan al momento de querer comprar un item
